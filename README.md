# 🍎 Exo_Fruits — Site E-commerce de Fruits & Légumes

> **Type :** Projet personnel  
> **Stack :** HTML, CSS, JavaScript, PHP  
> **Objectif :** Développer un site de vente en ligne avec gestion de catalogue et interface utilisateur complète  

---

## 🎯 Description

Exo_Fruits est une application web e-commerce simulant une boutique en ligne de fruits et légumes. Le projet couvre l'ensemble du cycle de développement web full-stack : conception de l'interface, affichage dynamique des produits et traitement des données côté serveur.

---

## ⚙️ Fonctionnalités

- 🛒 Catalogue de produits avec affichage dynamique
- 📦 Gestion des articles (ajout, consultation)
- 🖥️ Interface utilisateur intuitive et responsive
- ⚙️ Traitement des données côté serveur (PHP)
- 🗃️ Logique de navigation entre les pages

---

## 🛠️ Stack technique

| Technologie | Usage |
|-------------|-------|
| HTML | Structure et contenu des pages |
| CSS | Design et mise en page |
| JavaScript | Interactions et affichage dynamique |
| PHP | Logique serveur et traitement des données |
